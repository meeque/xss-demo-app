XSS Demo: Plain HTML Mock

This is a plain HTML page mock. This page is statically served and contains no client-side JS code either. Therefore, this page is not vulnerable to XSS by itself.

However, any other HTML document that is hosted on the same origin (protocol, host, port) can load this page and manipulate its DOM at will.
